jQuery

概念：也叫js库：就是一个js文件，存放了很多js的函数。jQuery把js中的Dom操作做了封装，可以快速查询使用里面的功能。

jQuery封装了js常会用的功能代码，优化了DOM、时间除了、动画设计、AJAX

原生js相当于楼梯，jQuery相当于电梯

优点：

  • 轻量级，文件才几十kb
  • 跨浏览器兼容
  • 链式编程、隐式迭代
  • 对事件、动画支持，大大简化了DOM操作
  • 支持插件扩展开发
  • 免费、开源

jquery的顶级对象$

相当于原生js中的window，可以用jQuery把元素用$包装成jQuery对象

jQuery和DOM对象

  1. 用原生js获取的对象就是DOM对象
  2. 用jQuery获取的对象就是jQuery对象
  3. jQuery本质是利用$对DOM对象包装后产生的对象(伪数组形式存储)。
  4. 两者可以相互转换。原生js比jquery更大，有些方法在jQuery里没有，要把$转换为DOM对象才能使用
    • dom转换为$ : $(DOM对象)
    • jQuery转换为DOM对象 index表示索引号，表示当前这个标签的顺序，从0开始
      • $('div')[index]
      • $('div').get(index)

一、jQuery选择器

选择器

  1. id选择器 $(“#id”)
  2. 全选选择器$(‘*’) 单引号！
  3. 类选择器 $(“.class”)
  4. 标签选择器 $(“div”)
  5. 并集选择器$(“div,p.li”)
  6. 交集选择器$(“li.current”)
  7. 子代选择器 $(“ul > li”) 选中一代儿子，不选孙子
  8. 后代选择器 $(“ul li”) 选中ul下的所有li元素，包括孙子等

隐式迭代

遍历内部DOM元素（伪数组形式存储）的过程就叫隐式迭代：给匹配到的所有元素进行循环遍历，不用我们再进行循环，方便调用

筛选选择器

:first 获取第一个 :last 获取最后一个 【单引号】

:eq(index) 获取索引为index的

:odd 获取索引号为奇数的 :even ……偶数的

siblings(selector) 找出所有兄弟 不包括自己

类操作和className区别

原生js中className会覆盖原先元素中的类名

jQuery相当于在原先的类名后面又加上一个类名，一个元素可有多个类名

jQuery的removeClass也只是删除相应的

三、jQuery效果

3.1显示隐藏效果

1.显示语法规范

show([speed,[easing],[fn]); //中括号表示可以不带这个参数

2.显示参数

  • 参数都可以省略，无动画直接显示
  • speed：”slow”,”normal”,”fast”,或者表示动画市场的毫秒数值
  • easing：用来指定切换效果 默认是swing，可用linear
  • fn：回调函数，在动画完成时执行的函数，每个元素执行一次

隐藏、切换与上述类似

一般情况下都不加参数，直接显示隐藏就可以了

3.2滑动

slideDown slideUp slideToggle

3.3 事件切换

hover([over,]out)

over：鼠标移到元素上要触发的函数

out：鼠标移出元素要触发的函数

3.4动画队列及其停止排队方法

stop(),停止排队，结束上一次的动画

3.5淡入淡出效果

fadeIn()

fadeOut()

fadeToggle()

利用渐进的方式调整到指定的不透明度 fadeTo([[speed],opacity,[easing],[fn]])

opacity 透明度必须写 0-1之间

speed也必须写

3.6 自定义动画 animate

animate（params，[speed],[easing],[fn]）

params：想要修改的样式属性 一对象形式传递，必须写，属性名可以不用带引号，如果是复合属性则需要采取驼峰命名法
