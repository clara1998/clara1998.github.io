个人博客搭建过程

1、安装npm

2、安装cnpm

3、安装hexo

输入命令 cnpm install -g hexo-cli

4、在git cmd窗口内克隆hexo

直接在windows的cmd窗口内输入命令不好用，要在git中输入命令 hexo init，等待一会儿

5、到这一步就搭建好了，启动网站页面

输入命令 hexo server，弹出网站页面如下

6.新建一篇文章

ctrl+c再输Y断开批处理操作

hexo n “我的第一篇博客文章”

输入命令hexo g 生成

再次启动得到：

7、部署到远端【搞了好半天】

准备工作：首先，在github上部署，新建一个仓库，再安装git部署插件输入命令：npm install –save hexo-deployer-git；

再设置myWeb文件夹下的_config.yml文件，windows系统直接用notepad++打开，在最下边添加deploy信息【每个冒号后边要加空格！！】

下一步，先打开git Bash，输入如下命令（给git配置身份）：

git config --global user.name "xxx" 其中xxx为github的用户名

git config --global user.email "xxx" 其中xxx为github的登录邮箱

接下来回到git CMD中，输入：

hexo d

这里的弹出一个选项：

我选择1，然后弹出一个网页，是git请求绑定github的请求，允许之后回到cmd窗口等待片刻后弹出如下信息后即部署成功！

此时刷新github的仓库，发现已经部署好了！（开心！！！）

现在只要在浏览器直接输入仓库名就可以访问啦！

8、自定义更换主题

https://github.com/litten/hexo-theme-yilia github上一位大佬做的主题，名字为yilia

以该主题为例，命令行输入

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

等待一会儿克隆完成，此时themes文件夹中有了yilia这个文件夹

再次修改config.yml文件，将theme改为yilia

然后清理一下，重新生成，最后打开，输入如下命令：

hexo clean

hexo g

hexo server

注意：这个时候还没有推到远端，还是要用localhost：4000来访问网站

然后再输入hexo d推到远端，就可以用仓库名直接访问了，主题更换完成~
